### docker简介

* **什么是Docker**
  * Docker 使用 Google 公司推出的 Go 语言 进行开发实现，基于 Linux 内核的 cgroup， namespace，以及 AUFS 类的 Union FS 等技术，对进程进行封装隔离，属于操作系统层面的虚拟化技术。
* **Docker的优势**
  * 更高效的利用系统资源
  * 更快速的启动时间
  * 一致的运行环境
  * 持续交付和部署
  * 更轻松的迁移
  * 更轻松的维护和扩展
